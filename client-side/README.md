# TestForge - Lifetime Offline Edition

## What is TestForge?

TestForge is a complete examination management platform for MSBTE-affiliated institutes. It handles everything from timetable creation to report generation.

## Quick Start (3 Steps)

### 1. Install Docker Desktop
Download and install from: https://www.docker.com/products/docker-desktop/

### 2. Run the installer

**Windows:**
- Double-click `install.bat` or `install.ps1`

**Linux / Mac:**
```bash
chmod +x install.sh
./install.sh
